# Mean-Variance-Standard-Deviation Calculator

This project is part of the FreeCodeCamp Data Analysis with Python certification.

## Description

This Python program uses NumPy to calculate statistical metrics from a list of 9 numbers.
The list is converted into a 3x3 matrix, and the following values are calculated:

- Mean
- Variance
- Standard Deviation
- Maximum
- Minimum
- Sum

Each metric is calculated across:
- Columns (axis=0)
- Rows (axis=1)
- Entire matrix (flattened)

- NumPy
